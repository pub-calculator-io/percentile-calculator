The percentile calculator helps to find percentile values for a data set. Use this percentile calculator to create a table listing each 5th percentile

[https://www.calculator.io/percentile-calculator/](https://www.calculator.io/percentile-calculator/)

== Usage ==

Add the Percentile Calculator shortcode to your page, post or sidebar:

`[ci_percentile_calculator]`

Add the following code to your template where you would like the Percentile Calculator to appear:

`<?php display_ci_percentile_calculator(); ?>`

== Installation ==

1. Upload the Percentile Calculator /ci_percentile_calculator/ folder to the /wp-content/plugins/ directory.
2. Activate the Percentile Calculator plugin through the "Plugins" menu in WordPress.
